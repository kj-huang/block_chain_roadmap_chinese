# 區塊鏈世界學習路線圖 🔗

這是一份給想進入區塊鏈世界的人的完整學習指南。無論你想寫智能合約、交易加密貨幣，還是運營挖礦與驗證節點，都能在這裡找到你的路線。

> 繁體中文 | 最後更新：2025 年

## 簡介

![區塊鏈世界 2018 原始路線圖](/img/blockchain2018.png)

*上圖為 2018 年的原始路線圖架構，以下內容已更新至 2025 年。*

區塊鏈世界可以分為三大圈子：

| 圈子 | 適合誰 | 做什麼 |
|------|--------|--------|
| **鏈圈** | 工程師、開發者 | 寫智能合約、建 DApp、參與協議開發 |
| **幣圈** | 投資者、交易者 | 交易加密貨幣、參與 DeFi、投資研究 |
| **礦圈** | 基礎設施營運者 | 挖礦、跑驗證節點、質押 |

三者相互交織 — 你可以同時涉足多個圈子。

## 目錄

### 📚 [基礎知識 — 進入任一圈子的必備功](docs/00-prerequisites.md)

不管你選哪條路線，這些是共通的基本功：

- **基本名詞** — 區塊、哈希、公私鑰、智能合約、Gas、Layer 1/2
- **共識機制** — PoW、PoS、DPoS、PBFT
- **記帳與資料結構** — UTXO vs Account 模型、區塊結構
- **P2P 技術** — Gossip Protocol、DHT、libp2p
- **商業知識** — 代幣經濟學、DAO、DeFi 概念、監管環境

### ⛓️ [鏈圈世界 — 技術開發路線](docs/01-chain-circle.md)

從零到一成為區塊鏈開發者：

```
程式語言 → 智能合約 → DApp 前端 → 公鏈生態 → 進階主題 → 安全審計
```

涵蓋：Solidity、Rust、Hardhat、Foundry、viem/wagmi、Layer 2、ZKP、帳戶抽象

### 💰 [幣圈世界 — 交易與投資路線](docs/02-coin-circle.md)

從買第一顆幣到深度投資研究：

```
錢包與交易所 → 交易知識 → DeFi 實戰 → NFT 與數位資產 → 投資研究 → 監管稅務
```

涵蓋：CEX/DEX、技術分析、鏈上分析、AMM、借貸、流動性質押、DYOR 方法論

### ⛏️ [礦圈世界 — 挖礦與驗證路線](docs/03-mining-circle.md)

從理解共識到運營基礎設施：

```
基礎概念 → PoW 挖礦 → PoS 驗證 → 節點運營 → 經濟分析 → 進階基礎設施
```

涵蓋：Bitcoin 挖礦、ETH 驗證者、流動性質押、節點架設、MEV、DePIN

### 📖 [參考資源總覽](docs/04-resources.md)

精選學習資源、開發工具、中文社群和求職平台。

```
                    ┌──────────────┐
                    │   基礎知識    │
                    │ (所有人必讀)  │
                    └──────┬───────┘
                           │
                    選擇你的方向
               ┌───────┼───────┐
               ▼       ▼       ▼
          ┌────────┐┌────────┐┌────────┐
          │ 鏈  圈 ││ 幣  圈 ││ 礦  圈 │
```

- **想了解投資**？從[幣圈](docs/02-coin-circle.md)開始
