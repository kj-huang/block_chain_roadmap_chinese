# 幣圈世界 — 交易與投資路線圖

> 幣圈是區塊鏈世界中最活躍的領域 — 這條路線適合想了解加密貨幣交易、DeFi 投資和資產管理的你。
> 從開設錢包到鏈上分析，從 DeFi 實戰到投資研究，建立你的交易知識體系。

## 圖例

| 標記 | 意義 |
|------|------|
| 🟠 | 個人推薦 — 強烈建議優先學習 |
| 🟡 | 建議學習 — 有助於全面理解 |
| ⬜ | 任君挑選 — 依個人興趣深入 |

> **風險警示**：加密貨幣市場波動極大，本路線圖僅為學習參考，不構成任何投資建議。請務必做好風險管理，只用你能承受損失的資金進行投資。

## Level 1：入門基礎

### 🟠 選擇交易所

| 類型 | 說明 | 範例 |
|------|------|------|
| CEX (中心化交易所) | 類似傳統券商，操作簡單，流動性佳 | Binance、Coinbase、OKX |
| CEX (台灣本地) | 支援台幣入金，符合台灣法規 | MAX、BitoPro、ACE |
| DEX (去中心化交易所) | 無需註冊，直接用錢包交易 | Uniswap、Raydium、Jupiter |

**新手建議**：先用台灣本地交易所以台幣購買 USDT，再轉至國際交易所或鏈上錢包操作。

### 🟠 錢包設定

| 類型 | 說明 | 推薦 |
|------|------|------|
| 瀏覽器錢包 | 安裝在瀏覽器中，方便與 DApp 互動 | MetaMask (EVM)、Phantom (Solana) |
| 行動錢包 | 手機 App，日常使用方便 | Trust Wallet、Rabby |
| 硬體錢包 | 離線儲存私鑰，安全性最高 | Ledger、Trezor |

**資產安全原則**：
- 🟠 助記詞用紙抄下，離線保存，絕對不要截圖或存在雲端
- 🟠 大額資產務必使用硬體錢包
- 🟡 啟用交易所的雙重驗證 (2FA)，優先使用 Google Authenticator
- 🟡 不要點擊來路不明的連結，小心釣魚網站和假空投

### 🟡 入金方式

| 方式 | 說明 |
|------|------|
| 法幣入金 | 透過台灣交易所以銀行轉帳購買 |
| P2P 交易 | Binance P2P 等平台，與其他用戶直接交易 |
| 穩定幣橋接 | 已有加密貨幣的情況下，透過跨鏈橋轉移 |

## Level 2：交易知識

### 🟠 現貨交易基礎

| 概念 | 說明 |
|------|------|
| 限價單 (Limit Order) | 指定價格買賣，等待成交 |
| 市價單 (Market Order) | 以當前最佳價格立即成交 |
| 止損單 (Stop Loss) | 價格跌破設定值時自動賣出 |
| 掛單簿 (Order Book) | 顯示所有買賣委託的列表 |
| 滑點 (Slippage) | 實際成交價與預期價格的差異（在 DEX 上尤其明顯） |

### 🟡 技術分析基礎

| 工具 | 說明 |
|------|------|
| K 線圖 (Candlestick) | 顯示開盤、收盤、最高、最低價 |
| 移動平均線 (MA) | 趨勢指標：MA20、MA50、MA200 |
| 支撐與阻力 | 價格反覆測試的上下邊界 |
| 成交量 (Volume) | 驗證價格走勢的可靠性 |
| RSI (相對強弱指標) | 判斷超買或超賣 |
| MACD | 趨勢動能指標 |

- 圖表工具：[TradingView](https://www.tradingview.com/)

### 🟡 鏈上分析

區塊鏈的透明性讓你可以直接觀察「聰明錢」的動向。

| 工具 | 用途 |
|------|------|
| [Dune Analytics](https://dune.com/) | 自訂 SQL 查詢鏈上資料，製作儀表板 |
| [Nansen](https://nansen.ai/) | 標記錢包身份，追蹤機構和巨鯨動態 |
| [Arkham](https://arkham.ai/) | 鏈上情報平台，追蹤資金流向 |
| [DeBank](https://debank.com/) | 查看任意地址的 DeFi 持倉 |
| [Etherscan](https://etherscan.io/) | 以太坊區塊瀏覽器，查交易和合約 |

### ⬜ 合約交易（期貨）

> **高風險警告**：合約交易涉及槓桿，可能導致全額虧損，新手務必謹慎。

| 概念 | 說明 |
|------|------|
| 永續合約 (Perpetual) | 沒有到期日的期貨合約 |
| 槓桿 (Leverage) | 借入資金放大倉位，同時放大盈虧 |
| 爆倉 (Liquidation) | 保證金不足時強制平倉 |
| 資金費率 (Funding Rate) | 多空雙方定期結算的費用 |
| 倉位大小 | 新手建議不超過總資金的 5-10% |

### ⬜ 選擇權基礎

- 買權 (Call) 與賣權 (Put)
- 權利金 (Premium)、行使價 (Strike Price)、到期日
- 平台：Deribit、Opyn

## Level 3：DeFi 實戰

### 🟠 DEX 交易與流動性提供

**自動做市商 (AMM) 原理**：
- 不用掛單簿，用流動性池 (Liquidity Pool) 和數學公式定價
- 恆定乘積公式：`x * y = k`
- 流動性提供者 (LP) 存入等值的兩種代幣，賺取交易手續費

| DEX | 鏈 | 特色 |
|-----|-----|------|
| Uniswap | Ethereum / L2 | AMM 始祖，最大的 DEX |
| Curve | Ethereum / L2 | 穩定幣交換專家，低滑點 |
| Jupiter | Solana | Solana 上最大的聚合交易平台 |
| Raydium | Solana | 結合 AMM 和訂單簿 |

**無常損失 (Impermanent Loss)**：
- LP 面臨的風險 — 當兩種代幣價格偏離時，你的資產價值可能低於單純持有
- 價格波動越大、越不對稱，無常損失越大
- 穩定幣對（如 USDC/USDT）的無常損失最小

### 🟠 借貸協議

| 協議 | 說明 |
|------|------|
| [Aave](https://aave.com/) | 最大的去中心化借貸平台 |
| [Compound](https://compound.finance/) | 借貸始祖之一 |
| [Morpho](https://morpho.org/) | 點對點借貸，利率更優 |

- **存款**：存入資產賺取利息（存款年化 APY）
- **借款**：超額抵押後借出其他資產
- **清算風險**：抵押品價值下降到清算線時，部分資產會被強制出售
- **閃電貸 (Flash Loan)**：在同一筆交易內借還的無擔保貸款

### 🟡 收益農場與質押

| 類型 | 說明 |
|------|------|
| 流動性挖礦 (Liquidity Mining) | 提供流動性獲得平台代幣獎勵 |
| 質押 (Staking) | 鎖定代幣參與網路驗證，獲得獎勵 |
| 收益聚合器 | 自動複利、尋找最佳策略（如 Yearn Finance） |
| 積分計畫 (Points) | 新協議用積分取代代幣獎勵，未來可能空投 |

### 🟡 穩定幣生態

| 類型 | 代表 | 機制 |
|------|------|------|
| 法幣抵押型 | USDT、USDC | 由發行方持有等值法幣儲備 |
| 加密資產超額抵押型 | DAI | 存入 ETH 等資產，超額抵押鑄造 |
| 演算法型 | FRAX | 部分抵押 + 演算法調節（UST 崩盤後此類風險較高） |

### ⬜ 衍生品協議

| 協議 | 說明 |
|------|------|
| [GMX](https://gmx.io/) | 去中心化永續合約交易，使用 LP 池作為對手方 |
| [dYdX](https://dydx.exchange/) | 訂單簿式永續合約（v4 遷移至 Cosmos 應用鏈） |
| [Hyperliquid](https://hyperliquid.xyz/) | 自建 L1 的去中心化永續合約 |

### ⬜ 流動性質押 (Liquid Staking)

| 協議 | 說明 |
|------|------|
| [Lido](https://lido.fi/) | 質押 ETH 獲得 stETH，可在 DeFi 中繼續使用 |
| [Rocket Pool](https://rocketpool.net/) | 更去中心化的 ETH 質押方案 |
| [Jito](https://jito.network/) | Solana 上的流動性質押 |

## Level 4：NFT 與數位資產

### 🟡 NFT 基礎

- 非同質化代幣 — 每個代幣都是獨一無二的
- 標準：ERC-721（單一）、ERC-1155（批量）
- 用途：數位藝術、遊戲道具、會員資格、門票、身份證明

### 🟡 NFT 交易市場

| 市場 | 說明 |
|------|------|
| [OpenSea](https://opensea.io/) | 最早且最大的 NFT 市場 |
| [Blur](https://blur.io/) | 專業交易者導向，交易速度快 |
| [Magic Eden](https://magiceden.io/) | 跨鏈 NFT 市場（Solana、Ethereum、Bitcoin） |

### ⬜ RWA (真實世界資產代幣化)

將傳統金融資產搬上區塊鏈的趨勢：
- 美國國債代幣化（如 Ondo Finance）
- 不動產代幣化
- 碳權代幣化
- 機構進場的重要推動力

## Level 5：投資研究

### 🟠 DYOR — 如何做自己的研究

「Do Your Own Research」是幣圈最重要的原則。以下是評估一個專案的框架：

| 面向 | 你應該問的問題 |
|------|--------------|
| 團隊 | 創辦人是誰？有沒有可查的背景？是否匿名？ |
| 技術 | 是否開源？程式碼品質？有沒有審計報告？ |
| 代幣經濟學 | 總量多少？怎麼分配？解鎖時程？有沒有賣壓？ |
| 社群 | Discord / Twitter 活躍度？真實互動還是機器人？ |
| 資金 | 融資了多少？投資者有誰？資金消耗速度？ |
| 產品 | 是否有可用的產品？用戶數？TVL？ |
| 敘事 | 是否搭上當前市場趨勢？競爭對手是誰？ |

### 🟡 數據工具

| 工具 | 用途 |
|------|------|
| [CoinGecko](https://coingecko.com/) | 代幣價格、市值、交易量 |
| [DefiLlama](https://defillama.com/) | DeFi 協議 TVL 追蹤 |
| [Token Terminal](https://tokenterminal.com/) | 協議營收和估值數據 |
| [Token Unlocks](https://token.unlocks.app/) | 代幣解鎖時程追蹤 |
| [CoinMarketCap](https://coinmarketcap.com/) | 市場總覽 |

### 🟡 風險管理

| 原則 | 說明 |
|------|------|
| 倉位控制 | 單一倉位不超過總資金的 10-20% |
| 分散投資 | 不要全押在一個代幣或一條鏈上 |
| 止損紀律 | 設定止損點並嚴格執行 |
| 分批進出 | 不要一次全買或全賣 |
| 牛熊意識 | 在牛市保持清醒，在熊市保持耐心 |

> **常見陷阱**：FOMO（害怕錯過）買在高點、槓桿過大、聽信 KOL 無腦跟單、不設止損。

## Level 6：監管與稅務

### 🟡 台灣監管現況

- 金管會將虛擬資產服務提供者 (VASP) 納入管理
- 交易所須完成洗錢防制 (AML) 法遵聲明
- 須遵循 KYC (認識你的客戶) 規範
- 目前無專法，以既有金融法規和洗錢防制法管理

### 🟡 稅務處理（台灣觀點）

| 情境 | 稅務處理 |
|------|----------|
| 交易獲利 | 目前台灣尚未對個人加密貨幣交易課徵資本利得稅 |
| 挖礦所得 | 可能視為營業所得或其他所得 |
| 空投收入 | 可能視為其他所得 |
| 海外交易所 | 須注意海外所得申報門檻 |

> **注意**：稅務規範可能變動，建議諮詢會計師或稅務專家。

### ⬜ 國際監管趨勢

| 地區 | 動態 |
|------|------|
| 美國 | SEC 積極監管，Bitcoin / Ethereum ETF 已獲批准 |
| 歐盟 | MiCA 框架實施，為加密資產提供統一監管 |
| 香港 | 開放散戶交易，推動 Web3 發展 |
| 新加坡 | MAS 發牌監管，對散戶保護嚴格 |
| 日本 | 最早立法，持牌交易所制度成熟 |
